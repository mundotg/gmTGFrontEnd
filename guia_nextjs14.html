📘 Guia Completo: Next.js 14

🚀 Criar Projeto

npx create-next-app@latest my-app

📂 Estrutura

my-app/
├── app/                # App Router (Next 13+)
│   └── page.tsx        # Página principal
├── public/             # Arquivos estáticos
├── styles/             # Estilos globais
├── next.config.js      # Configuração Next.js
├── tsconfig.json       # Configuração TypeScript
├── package.json
└── .env.local

⚙️ Configurações

next.config.js:

/** @type {import('next').NextConfig} */
const nextConfig = {
  reactStrictMode: true,
  experimental: {
    serverActions: true,
    appDir: true,
  },
}

module.exports = nextConfig;

🔌 Variáveis de Ambiente

# .env.local
DATABASE_URL=postgres://user:pass@localhost:5432/db

💾 Fetch de Dados (Server)

// app/page.tsx
async function getData() {
  const res = await fetch("https://api.exemplo.com")
  return res.json()
}

export default async function Page() {
  const data = await getData()
  return <pre>{JSON.stringify(data)}</pre>
}

🔒 Middleware

// middleware.ts
import { NextResponse } from 'next/server'

export function middleware(request) {
  return NextResponse.next()
}

export const config = {
  matcher: ['/api/:path*', '/dashboard/:path*'],
}

💡 Comandos Úteis

  • npm run dev – iniciar servidor local
  • npm run build – build de produção
  • npm start – iniciar app em produção
  • npm install pacote – instalar pacote

📦 Pacotes Úteis

  • tailwindcss – estilos rápidos
  • next-auth – autenticação
  • prisma – ORM para banco de dados
  • axios – requisições HTTP
